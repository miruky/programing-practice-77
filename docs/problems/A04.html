A04

10進数を2進数へ

整数を 10 桁の 2 進法表記で出力する。
章 1章: アルゴリズムと計算量
主な技法
ビット進法
原典 GitHub で原文を見る
A04.py ─ Python 3 
# =====================================================================
# A04: 10進数を2進数へ
# ---------------------------------------------------------------------
# 【概要】0 以上 1023 以下の整数 N を 10 桁の 2 進数表記で出力する。
# 【アルゴリズム】2^x で割って 2 で割った余りを取り、上位桁から並べる。
# 【計算量】O(log N) = O(10)。
# 【Pythonic 別解】 `print(format(N, "010b"))` で同じ結果が得られる。
# =====================================================================

# --- 入力 -----------------------------------------------------------
N = int(input())

# --- 各桁を上から下へ計算 -------------------------------------------
# x = 9, 8, ..., 0 の順に "2^x の位" を取り出して連結出力する。
#   wari = 2^x として N を wari で割った商の偶奇 (=その桁のビット) を求める。
# print の end='' により改行せず連結出力する。
for x in [9, 8, 7, 6, 5, 4, 3, 2, 1, 0]:
    wari = (2 ** x)
    print((N // wari) % 2, end='')

# 最後に改行する (end='' で改行を抑制しているため明示的に必要)。
print("")

N = int(input())

for x in [9, 8, 7, 6, 5, 4, 3, 2, 1, 0]:
    wari = (2 ** x)
    print((N // wari) % 2, end='')

print("")
