# =====================================================================
# C01: Tax Rate (力試し問題)
# ---------------------------------------------------------------------
# 【概要】消費税率 10% のとき、税抜価格 N 円の税込価格 (1.1N 円) を出力する。
#         制約より N は 100 の倍数なので、1.1N は必ず整数になる。
# 【アルゴリズム】N * 11 // 10 を計算するだけ。
# 【計算量】O(1)。
# 【ポイント】Python は浮動小数で計算すると精度の問題が出るので、
#             N * 11 // 10 のように整数演算で書くのが安全。
# =====================================================================

# --- 入力 -----------------------------------------------------------
N = int(input())

# --- 出力 (1.1N を整数で計算) ---------------------------------------
print(N * 11 // 10)

N = int(input())

print(N * 11 // 10)
