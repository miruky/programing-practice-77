C10: A Long Grid (力試し問題)

概要

2 行 W 列のグリッドを正しく塗り分ける方法数を mod 10⁹+7 で求める。 隣接 (左右・上下) は異なる色でなければならない (色は 4 色)。

アルゴリズム

数学的解析: 1 列目の塗り方は 4 × 3 = 12 通り。それ以降の 各列は 7 通り (前列に依存)。よって答え = 12 · 7^(W-1)。 7^(W-1) は繰り返し二乗法で O(log W)。

計算量

O(log W)。W = 10⁹ でも約 30 回ループで終わる。

# a^b mod m を返す関数 (繰り返し二乗法)
def Power(a, b, m):
    p = a
    Answer = 1
    for i in range(60):
        wari = 1 << i
        if (b // wari) % 2 == 1:
            Answer = (Answer * p) % m
        p = (p * p) % m
    return Answer

# --- 入力 -----------------------------------------------------------
W = int(input())

# --- 出力 -----------------------------------------------------------
print(12 * Power(7, W - 1, mod) % mod)
