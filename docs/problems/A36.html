# =====================================================================
# A36: ちょうど K 歩 (偶奇考察)
# ---------------------------------------------------------------------
# 【概要】N×N の盤面で、左上 (1,1) から右下 (N,N) まで丁度 K 歩で
#         移動できるかを判定する。
# 【アルゴリズム】考察 (上界 + 偶奇)
#   - 最短歩数は 2N-2。これより K が小さければ不可能。
#   - 1 歩進む → 戻るで歩数を 2 ずつ増やせるので、K - (2N-2) は偶数。
#   ⇔ K >= 2N-2 かつ K と 2N-2 の偶奇が同じ (= K が偶数)
# 【計算量】O(1)
# =====================================================================

N, K = map(int, input().split())

# 必要十分条件: K >= 最短歩数 かつ K が偶数
if K >= 2 * N - 2 and K % 2 == 0:
    print("Yes")
else:
    print("No")

N, K = map(int, input().split())

if K >= 2 * N - 2 and K % 2 == 0:
    print("Yes")
else:
    print("No")
