基本問題 A26: 素数判定 試し割り法で素数かを判定する。計算量 O(√N)。
B26.py ─ Python 3 
# =====================================================================
# B26: Output Prime Numbers (A26 応用 ─ エラトステネスのふるい)
# ---------------------------------------------------------------------
# 【概要】N 以下の素数を昇順にすべて出力する。
# 【アルゴリズム】エラトステネスのふるい。
#         2 から √N までの各 i について、「まだ消されていなければ」その
#         倍数 2i, 3i, ... を消していく。残った数が素数。
# 【計算量】O(N log log N)。N = 10⁶ までほぼ瞬時。
# =====================================================================

# --- 入力 -----------------------------------------------------------
N = int(input())

# --- ふるいの本体 --------------------------------------------------
# Deleted[x] = True ⇔ x は素数でない
Deleted = [False] * 1000009
LIMIT = int(N ** 0.5)
for i in range(2, LIMIT + 1):
    if not Deleted[i]:
        # i 自身は素数。その倍数 (2i, 3i, ...) を全て消していく。
        for j in range(i * 2, N + 1, i):
            Deleted[j] = True

# --- 残った数を素数として出力 --------------------------------------
for i in range(2, N + 1):
    if not Deleted[i]:
        print(i)

N = int(input())

    if not Deleted[i]:
        for j in range(i * 2, N + 1, i):
            Deleted[j] = True

for i in range(2, N + 1):
    if not Deleted[i]:
        print(i)
