B45.py ─ Python 3 
# =====================================================================
# B45: Blackboard 2 (A45 応用 ─ 不変量)
# ---------------------------------------------------------------------
# 【概要】3 つの整数 (a, b, c) に対して「2 つを選んで両方に +1 / 両方に -1」
#         のような操作を繰り返すと、総和 a+b+c が不変。よって、最終的に
#         全部 0 にできる ⇔ 最初から a+b+c = 0 のときに限る。
# 【アルゴリズム】総和を取って 0 か判定するだけ。
# 【計算量】O(1)。
# 【教訓】操作のもとで保たれる量 (不変量) を見抜くと、答えが一発で決まる。
# =====================================================================

# --- 入力 -----------------------------------------------------------
a, b, c = map(int, input().split())

# --- 出力 -----------------------------------------------------------
print("Yes" if a + b + c == 0 else "No")

a, b, c = map(int, input().split())

print("Yes" if a + b + c == 0 else "No")
