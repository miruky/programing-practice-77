勝ち負け状態を DP で後退解析する。

# =====================================================================
# A32: 後退解析ゲーム
# ---------------------------------------------------------------------
# 【概要】石が N 個。各手番で A 個か B 個取れる。取れない方が負け。
#         先手必勝なら First, 後手必勝なら Second を出力する。
# 【アルゴリズム】後退解析 DP
#   dp[i] = "残り i 個のとき、手番の人が勝てるか"
#   dp[i] = True ⇔ "相手を負け状態 (dp[?] = False) に追い込める手がある"
# 【計算量】O(N)
# =====================================================================

# --- 入力 -----------------------------------------------------------
N, A, B = map(int, input().split())

# dp[i] = True ⇔ "残り i 個で手番の人が勝てる"
dp = [None] * (N + 1)

# 小さい方から決める (後退解析: 終端 0 が負け状態, dp[0]=False)
for i in range(N + 1):
    if i >= A and dp[i - A] == False:
        # A 個取って相手を負けに追い込めるなら勝ち
        dp[i] = True
    elif i >= B and dp[i - B] == False:
        # B 個取って相手を負けに追い込めるなら勝ち
        dp[i] = True
    else:
        # どちらの手も相手の負けにならない → この状態は負け
        dp[i] = False

# --- 出力 -----------------------------------------------------------
if dp[N] == True:
    print("First")
else:
    print("Second")
