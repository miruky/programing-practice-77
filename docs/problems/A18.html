部分和問題

選んだカードの合計を S にできるかを 2 次元 DP で判定する。

# =====================================================================
# A18: 部分和問題
# ---------------------------------------------------------------------
# 【概要】N 枚のカード (値 A[i]) から、合計を S にできるかを判定する。
# 【アルゴリズム】2 次元 DP
#   dp[i][j] = 最初の i 枚から合計 j を作れるか (True/False)
#   遷移: dp[i][j] = dp[i-1][j]  (i 枚目を選ばない)
#         dp[i][j] = dp[i-1][j-A[i-1]]  (i 枚目を選ぶ; j>=A[i-1])
# 【計算量】O(NS)
# =====================================================================

# --- 入力 -----------------------------------------------------------
N, S = map(int, input().split())
A = list(map(int, input().split()))

# --- DP テーブル初期化 ----------------------------------------------
# dp[i][j] : i 枚から合計 j を作れるか
dp = [[None] * (S + 1) for _ in range(N + 1)]

# 0 枚で 0 は作れる (空和)、それ以外は不可。
dp[0][0] = True
for i in range(1, S + 1):
    dp[0][i] = False

# --- DP 遷移 -------------------------------------------------------
for i in range(1, N + 1):
    for j in range(0, S + 1):
        if j < A[i - 1]:
            # j が A[i-1] より小さい → i 番目は選べない
            dp[i][j] = dp[i - 1][j]
        if j >= A[i - 1]:
            # 選ぶ・選ばないの両方を試して OR を取る
            if dp[i - 1][j] == True or dp[i - 1][j - A[i - 1]] == True:
                dp[i][j] = True
            else:
                dp[i][j] = False

# --- 出力 -----------------------------------------------------------
if dp[N][S] == True:
    print("Yes")
else:
    print("No")
