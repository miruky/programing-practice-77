A17

ダンジョン2(復元)

DP で求めた最短距離をもとに、経路を復元する。
章 4章: 動的計画法
主な技法
DP経路復元
原典 GitHub で原文を見る
A17.py ─ Python 3 
# =====================================================================
# A17: ダンジョン2 (DP + 経路復元)
# ---------------------------------------------------------------------
# 【概要】A16 と同じ問題で、さらに最短経路 (通過した部屋の列) を求める。
# 【アルゴリズム】DP 後に逆方向にトレースバックする経路復元の典型手法。
#   ゴールから出発し、dp の値と一致する遷移をたどり直す。
# 【計算量】O(N)
# =====================================================================

# --- 入力・DP (A16 と同じ) -----------------------------------------
N = int(input())
A = list(map(int, input().split()))
B = list(map(int, input().split()))

dp = [None] * (N + 1)
dp[1] = 0
dp[2] = A[0]
for i in range(3, N + 1):
    dp[i] = min(dp[i - 1] + A[i - 2], dp[i - 2] + B[i - 3])

# --- 経路復元 ------------------------------------------------------
# Place は現在位置。ゴール N から始めて部屋 1 まで遡る。
# 例: 入力例 1 の場合、Place は 5 → 4 → 2 → 1 と変化する。
Answer = []
Place = N
while True:
    Answer.append(Place)
    if Place == 1:
        break
    # 「どちらの遷移で来たか」を dp 値から逆算する。
    # dp[Place-1] + A[Place-2] == dp[Place] ならば 1 個前から来た。
    if dp[Place - 1] + A[Place - 2] == dp[Place]:
        Place = Place - 1
    else:
        Place = Place - 2
Answer.reverse()  # スタート→ゴール の順に並べ直す

# --- 出力 -----------------------------------------------------------
Answer2 = [str(i) for i in Answer]
print(len(Answer))
print(" ".join(Answer2))
