法則 A55

順序付き集合 — 値の挿入/削除と「以上/以下」検索

C++ の std::set 相当。Python では SortedSet ライブラリ

「値の挿入、削除、x 以上の最小要素、x 以下の最大要素」を 全て O(log N) で行えるデータ構造。C++ の std::set が標準的だが、Python の標準ライブラリには無いので外部ライブラリ (SortedSet) を使う。

「最も近い値」「順位」「上位 K 個」などのクエリで威力を発揮。難問の構築テクとして頻出する一方、初学者には実装の選択肢が乏しいのが Python の悩み。

Step 1: できること

  • 挿入: s.add(x) / 削除: s.discard(x) — O(√N) or O(log N)
  • x 以上の最小: s.ge(x) — O(√N)
  • x 以下の最大: s.le(x) — O(√N)
  • 順位: s.index(x)
flowchart LR
  Q[クエリ: x] --> G[ge x: 以上の最小]
  Q --> L[le x: 以下の最大]
  Q --> D["差 = min(x-le, ge-x)"]

Step 2: Python での選択肢

  • SortedSet ライブラリ (tatyam-prime/SortedSet): 平方分割ベース、O(√N)
  • sortedcontainers.SortedSet: B 木ベース、AtCoder にプリインストール
  • 自作 BIT で番兵 (高度)

本サイトでは SortedSet ライブラリを採用 (B55)。

Step 3: 番兵テクニック

「x より小さい数がない」「x より大きい数がない」のケース回避に、-INF と +INF を最初から入れておく と分岐が消える。

詰まりやすい点

  • Python 標準にはない。AtCoder では sortedcontainers が使える。
  • 重複を扱いたいなら SortedList (set ではなく list の順序版) を使う。
  • 番兵を入れたら、「実要素ゼロ」の判定を len(s) == 2 で行う。
