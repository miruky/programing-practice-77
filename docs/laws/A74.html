法則 A74

転倒数分解 — スライドパズルを行・列に分ける

2 次元の問題を 2 つの 1 次元問題に

スライドパズル: N×N 盤面の数を目標配置にする最小スワップ数。操作が 行内 / 列内に閉じている とき、行の並べ替えと列の並べ替えに独立分解 できる。

各方向の操作回数 = 転倒数 (i < j で A[i] > A[j] となる組数)。1 次元 2 つの転倒数の和が全体の答え。

Step 1: 行・列に分解

盤面から、行 X[i] と列 Y[j] の代表値を抽出。「行同士の並べ替えと列同士の並べ替えに独立に分解できる」性質を使う。

flowchart LR
  G[2 次元盤面] --> X[行の代表値 X]
  G --> Y[列の代表値 Y]

Step 2: 各列の転倒数を計算

X と Y のそれぞれについて、転倒数を素朴な二重ループで計算 (N が小さければ O(N²) で OK)。大きいなら マージソート / BIT で O(N log N)。

flowchart LR
  X[X] --> IX[転倒数 X]
  Y[Y] --> IY[転倒数 Y]

Step 3: 和が答え

answer = 転倒数(X) + 転倒数(Y)。 操作が独立だから単純な和になる。

詰まりやすい点

  • 操作が行内・列内に閉じているという前提を確認 (一般のスライドパズルには使えない)。
  • 転倒数の効率計算 = BIT (Fenwick) + 座標圧縮 (A15) のコンボ。
  • 0 マスがある場合、扱いを正しく決める。
